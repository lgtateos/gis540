How to submit the project

Final projects sometimes exceed the space available on Wolfware for submission, so we have set up a special server for this purpose. The folder is named XYYYY (where X==S or F (for Spring or Fall) and YYYY == the current year)To submit your project:

IF you are using a CNR lab machine:
   Type \\cnrdata.cnr.ncsu.edu\geospatial$\Education\GIS540\projects\XYYYY in a Windows Explorer window.

ELSE (you not using a CNR lab machine):
    Connect through a virtual private network (VPN) to get access to this directory. To do this login at vpn.ncsu.edu.
    Next type\\cnrdata.cnr.ncsu.edu\geospatial$\Education\GIS540\projects\XYYYY in a Windows Explorer window
For the username type: WOLFTECH\unityid and use your unity password.

--For the Project Press Release, submit your text document in the ABSTRACTS directory and submit your thumbnail image in the THUMBNAILS directory.

  1. For the project release, submit a text document named as your unity ID. E.g., The instructor's document would be named, lgtateos.txt. Format this document as shown in this example. Use the words Title, Name, Synopsis, and Keywords with colons behind them. Run this script on your text file to see how your html gallery entry will look. Submit this text file in this directory: \\cnrdata.cnr.ncsu.edu\geospatial$\Education\GIS540\projects\XYYYY\ABSTRACTS

    It must containing the following:
    1. Title: A short title indicating the topic.
    2. Name: Your name as it will appear in the project gallery.
    3. Synopsis: up to 4 sentences telling what your project does. (This will be posted with your title, thumbnail and links to your presentations in the course gallery.) Use a spelling-checker!
    4. Keywords: Comma separated list of keywords relating to the project. These should include topic area, data types, and tools. This is meant to help other students search for related project topics.

  2. Submit a thumbnail image for the project gallery. Name it as your unity ID. E.g. J.K Rowling's picture would be named jkrowlin.jpg Submit this picture in this directory \\cnrdata.cnr.ncsu.edu\geospatial$\Education\GIS540\projects\XYYYY\THUMBNAILS.

    Image requirements:
    -- format: JPEG, PNG, or GIF image file
    -- 62 pixels wide and <=62 pixels tall. (The latest version of Windows Programs > Accessories > Paint allows you to resize images.)
    -- Select a colorful image that represents the topic of your project.
    -- See project gallery for examples

---Place all other project materials in your userid subdirectory as described on the "Submission Format" page. If you don't see one with your id, let us know. Place a file in your project directory now to confirm that you have access to it.   You can add or delete files, just as you would to any directory where you have write access, so you should upload incomplete versions of your project ahead of time.
