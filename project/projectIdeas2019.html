Potato Blight Project

This project involves working with data extracted from documents written in the mid-1800s, during the potato blight outbreak that led to the Irish Potato Famine. The disease appeared in the US prior to appearing in Ireland.  The pathogen that caused the disease is known, but how the disease spread through the US is still uncertain. To learn more about the patterns of spread, researchers are using natural language processing to study historical agricultural documents and extract passages from these that contain both mentions of potato disease and mentions of geographic locations. This project aims to make maps out of the data that has been collected through this procedure. See the extended description and sample data linked here.

Pokemon Go

The tables provided with this project are representative of information you can download about the game.   Your challenge would be to map and analyze this type of data.   Can you think of any reason why NSA analysts might be interested in studying data like this?   You can make a tool that searches for patterns in the data like, which Pokemon characters have more compact location sightings than others?  What areas (if any) do the Pokemon characters avoid?  For each character, are do they show up near roads or streams?  Is there a particular land cover Pokemon characters appear in?   Are there correlations between appearances (do certain Pokemon characters show up near each other consistently)? For most of these questions, you can ask analogous ones about Pokestops.   Can you come up with other interesting questions to answer?

Eye tracker project

An eye tracker is a device that tracks the locations of a viewer’s gaze dwells (fixations) and the movements from one fixation to another (saccades) on an electronic display.  The eye tracker device in the NCSU GIS Visualization Lab can be used with ArcMap to gather data that specifies where on a map a user is focusing as well as when and how long their gaze lingers.  The eye tracking data is saved to a file in comma separated value (csv) format. In this project you'll be performing analysis on this spatial data. For more info, click here...

Interagency Compliance Application

The National Park Service is responsible for identifying properties and points of interest that may be impacted due to anthropogenic activities.  For example, if an abandoned industrial site is to be converted to a wind farm, are there any National Natural Landmarks that could be visually impaired?  Additionally, if a county is scheduled to change zoning regulations, are there any Federal lands in the county that could potentially be affected?  The application is not designed for individual compliance applications, e.g.  viewshed or environmental impact assessments, but rather to identify potentially impacted properties.
